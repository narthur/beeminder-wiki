Interfaces for Beeminder #

Things that (not only) automate data entry, but read from Beeminder and offer an Interface for it. Thanks to @philip for his awesome list, which I basically duplicated here.

P. S.: Searching for the topics beeminder or beeminder-api on GitHub might yield more projects to try out.

  • beeminder/beebot - Slack-Bot by @apb

  • lydgate/bmndr written in Python - by @bkam
  • pjjh/bmndr written in Perl (was originally a fork of lydgate/bmndr, which originally was a perl implementation) - by @philip
  • malcolmocean/beeminderjs written in NodeJS - by @malcolm

Web #

  • Beeminder Autodialer - by @narthur. "The Beeminder autodialer will automatically adjust the rate on your goals based on your historical performance."
  • Beescheduler - by @enolan. "Beescheduler lets you schedule different rates for your Beeminder goals based on the days of the week. I wrote it so I could focus on one project Monday - Wednesday and different ones on Thursday and Friday. This is much better than doing a tiny amount on all of them every day." Forum announcmenet.
  • chipmanaged/MCM-Dashboard - by @mary, includes an autodial feature and umbrella goals feature, see this thread
  • Altbee - "an alternative web interface for Beeminder." Announcement thread. Source code.
  • beeminder-to-sqlite - by Ben Congdon, meant for viewing your Beeminder data in Datasette.
  • BUI - "an alternative web interface for Beeminder but with more charts". Source code. Announcement thread
